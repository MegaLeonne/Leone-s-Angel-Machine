# Leone's Angel Machine - Documentation

This directory contains the core lore, philosophy, and structured data of the Angel Machine.

## Directory Structure

- [Philosophy](./philosophy/): Cosmology, Abyss, and Void Tongue.
- [Archetypes](./archetypes/): Head, Heart, and other conceptual entities.
- [Rituals](./rituals/): Invocations and ceremonial procedures.
- [Guides](./guides/): Contribution and system usage instructions.
- [Indices](./index/): Centralized lookup and taxonomy files.
- [Orphans](./orphans/): Fragmented lore and unidentified records.
